Computertimetracker

A program that tracks time spent on the computer. I made this to assist me in my daily hour reporting.

View project onGitHub

The Computer Time Tracker program tracks time spent on the computer, so the user can track their daily work time.

The program tracks user session events (logon, lock, unlock) and divides computer usage to periods between such events. The user can exclude certain periods from the calculated work time.

The program should be running whenever the user is logged on, so it should be launched at logon by adding it in the Startup folder. When running, it is visible on the notification area. The program recognizes when it is started for the first time on a day and starts calculating computer usage time from then on.
